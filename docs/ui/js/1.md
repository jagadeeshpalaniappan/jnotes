## 2. Remove duplicates from Array (OneLiner)

```js
let inputNums = [1,2,2,3];

// Array to Set
let set = new Set(inputNums);
console.log(set); // 1,2,3

// Set to Array:
console.log([...set]); // [1,2,3]

// OneLiner:
console.log([...new Set(inputNums)]); // [1,2,3]
```
